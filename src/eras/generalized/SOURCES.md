# Era 3 (Generalized Learning) — Source Attributions

Tracked here so they can move to an "About" page later. The header only shows the three top-level sources (Wikipedia, Common Crawl, GPT-2); everything else lives here.

## Datasets / model outputs

- **Wikipedia** — https://en.wikipedia.org/ (CC BY-SA 4.0)
  - Stage 1 (Predict): snippets from Cat, Apollo 11, Mount Everest, Albert Einstein, Mount Kilimanjaro, Eiffel Tower, Pizza, Venus, Banana.
- **Common Crawl sample** — https://huggingface.co/datasets/agentlans/common-crawl-sample
  - Stage 1: casual/web fragments (running, surf, grammar, NPR headlines, NAND).
- **GPT-2 (OpenAI, 2019)** — https://huggingface.co/openai-community/gpt2
  - Stage 3 (Tag): real/representative next-word continuations across all pairs.
- **Project Gutenberg** — public domain
  - Hamlet (https://www.gutenberg.org/files/1524/1524-h/1524-h.htm), Huck Finn (https://www.gutenberg.org/files/76/76-h/76-h.htm).
